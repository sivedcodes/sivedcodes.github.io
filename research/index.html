CVE-2026-0098 — Android Version Spoofing & Silent Installation

Discovered: Feb 2026 · Patched by Google: May 2026

Critical vulnerability in Android's PackageManager allowing attackers to spoof version codes and trigger silent installations. Root cause: missing validation of versionCode against signing certificates during INSTALL_PACKAGES Intent resolution.

Impact: Arbitrary code execution via malicious "update" delivery without user consent

OTP Bypass System Vulnerability

Discovered: 2017

Discovered a critical flaw in SMS-based OTP authentication. The server generated session tokens before OTP verification completed, allowing direct session token access without valid OTP. API-level session manipulation enabled complete account takeover.

Impact: Full account compromise — personal data, transactions, sensitive operations
